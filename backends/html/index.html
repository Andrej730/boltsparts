BOLTS

BOLTS is an Open Library of Technical Specifications.

Attention: BOLTS is beta software

BOLTS is in its early stages of development, so expect things to change. But this is also your chance to make things change for the better.

What is BOLTS?

The aim of BOLTS is to build a free and open-source standard parts library for CAD applications. It does so by providing a common database that can be utilized by many different CAD applications.

BOLTS tries to provide a infrastructure and processes that build on three basic principles:

  • Separation: The description of parts (dimensions, geometries, ...) is separated into application specific and general information. This allows to add support for individual CAD applications with minimal effort.
  • Automation: The description of parts is designed to be machine readable. This allows to eliminate a lot of work by automation, and eases the task of maintaining a large number of parts.
  • Contribution: Building a parts library that suits the needs of a wide range of people can only be done by a wide range of people. Therefore BOLTS is designed to make contributing easy.

Features

  • Provides GPL and LGPL distributions.
  • Built with platform independent technologies like Python and YAML
  • Support for OpenSCAD and FreeCAD, others are planned.
  • Comprehensive online documentation for all parts
  • Easy to contribute to

Quickstart: Using BOLTS with OpenSCAD

Download the tarball with BOLTS for OpenSCAD from the Downloads page and extract it into a folder. Start OpenSCAD from this folder or Adjust your OpenSCAD library path to make sure OpenSCAD can find BOLTS. Then add this at the top of your scad file:

{% highlight python %} include {% endhighlight %}

You can now use BOLTS to add standard parts like a washer:

{% highlight python %} ISO7089("M4"); {% endhighlight %}

For more information see the Documentation

Quickstart: Using BOLTS with FreeCAD

Download the tarball with BOLTS for FreeCAD from the Downloads page and extract it into the FreeCAD macro directory. If you don't know where this is, you can look it up in the preferences dialog on the macro tab. You also have to make sure to have pyyaml installed.

After starting FreeCAD, choose Macros from the Macro menu and execute the start_bolts.FCMacro

A widget should appear. There you can select a part from the tree, give values for the parameters and add it to the current document.
