Bases

For some classes a backend specific geometry is missing for one or more CAD packages. Detailed information how to add a geometry can be found here for OpenSCAD, and here and here for FreeCAD. $missingbasetable

Drawings

For some classes a drawing is missing that gives a visual overview over the part and clarifies the meaning of the parameters. Instructions how to add one can be found here $missingdrawings

For other classes no svg version of the drawing is available. $missingdrawingssvg

Common Parameters

For some classes no common parameter combinations are specified. This prevents these classes from being exported in some backends that create individual files for a combination of parameters. $missingcommonparameters

Unsupported licenses

For some classes a unsupported license name and/or url was found. Often this is the result of typos. Check with the list in the specifications to identify these cases. If the license is not on the list, try contact all the copyright holder of the file and ask them to relicense or dual license this file under one of the supported licenses, or remove the file. $unsupportedlicenses
